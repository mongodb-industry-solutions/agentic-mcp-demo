# Privacy Policy — Agent Live

**Last updated: May 2026**

Agent Live is a log viewer for live AI agent activity streams.

## Data Collection

Agent Live does **not** collect, store, or transmit any personal data. Specifically:

- No account or registration is required
- No analytics or tracking libraries are used
- No device identifiers, location data, or usage statistics are collected
- No data is shared with third parties

## Network Activity

The app connects to a server-sent event (SSE) endpoint to receive and display real-time log messages. This connection is read-only. No data originating from the device is sent to that endpoint.
